# 🎓 AI & ANN Tech Talk 2K26 — Certificate Verification System

A web-based certificate verification platform developed for the **AI & ANN Tech Talk 2K26**, organized by IEEE HIT Student Branch.

This system enables participants to verify the authenticity of their certificates using a unique Certificate ID, ensuring transparency and credibility.

## 📌 How It Works

1. User enters a valid Certificate ID (e.g., `AI26-XXXX`)
2. Frontend sends a request to the API
3. Backend verifies the ID from Google Sheets
4. System returns:
   - ✅ Verified (Attended)
   - ⚠️ Not Attended
   - ❌ Invalid / Not Found
5. Result is displayed instantly on the UI

---

## ⚠️ Important Note

- Certificates are issued **only to attendees** of the event  
- If you did not attend, your certificate was not generated  
- Invalid or unknown Certificate IDs will not return a valid result
